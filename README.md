## ðŸ˜Š How to run

Before starting, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed on your machine.

```sh
- Clone this repo:
$ git clone https://github.com/marcelopajr/webdev-next.git

- Enter directory:
$ cd webdev-next

- Install dependencies:
$ yarn

- Launch the app:
$ yarn dev
```
